# Créer un compte GitHub

GitHub est une plateforme d'hébergement de code basée sur Git. Elle permet de stocker, partager et collaborer sur des projets de développement.

## Prérequis

- Une adresse e-mail valide.
- Un navigateur web.
- Une connexion Internet.

## Étapes

### 1. Accéder au site GitHub

Rendez-vous sur :

https://github.com

### 2. Créer un compte

Cliquez sur **Sign up**.

### 3. Renseigner les informations

Complétez les champs demandés :

- **Email** : votre adresse e-mail.
- **Password** : un mot de passe sécurisé.
- **Username** : votre nom d'utilisateur GitHub.

### 4. Vérifier votre adresse e-mail

GitHub envoie un e-mail de confirmation. Ouvrez-le puis cliquez sur le lien de validation.

### 5. Finaliser l'inscription

Suivez les dernières étapes proposées par GitHub jusqu'à l'accès à votre tableau de bord.

## Vérification

Une fois connecté, votre profil est accessible depuis l'icône située en haut à droite de la page.

## Notes

- **Notes :**
  - Choisissez un nom d'utilisateur professionnel si vous comptez partager vos projets.
  - Utilisez un mot de passe fort et unique.
  - Activez l'authentification à deux facteurs (2FA) pour renforcer la sécurité de votre compte.
