# Installation de Visual Studio Code (VS Code)

Visual Studio Code est un éditeur de code source léger et puissant développé par Microsoft. Il est largement utilisé pour le développement web et logiciel.

---

## Prérequis

- Une connexion Internet
- Un ordinateur (Windows, macOS ou Linux)

---

## Installation sur Windows

### 1. Télécharger VS Code

Rendez-vous sur le site officiel :

https://code.visualstudio.com

Cliquez sur **Download for Windows**.

### 2. Installer VS Code

Lancez le fichier téléchargé et suivez les étapes :

- Acceptez les conditions
- Laissez les options par défaut
- Cochez l’option **Add to PATH** (important)
- Cliquez sur **Install**

### 3. Lancer VS Code

Après installation, ouvrez Visual Studio Code depuis le menu Démarrer.

---

## Installation sur macOS

### 1. Télécharger VS Code

https://code.visualstudio.com

### 2. Installer

- Ouvrez le fichier téléchargé
- Glissez VS Code dans le dossier **Applications**

---

## Installation sur Linux (Ubuntu/Debian)

### Méthode via Snap

```bash
sudo snap install code --classic
```

---

## Vérification

Vous pouvez vérifier l’installation en ouvrant un terminal et en tapant :

```bash
code --version
```

---

## Extensions recommandées

Pour bien débuter, installez ces extensions :

- ESLint
- Prettier
- GitLens
- PHP Intelephense (si vous utilisez PHP/Laravel)

---

## Notes

- **Notes :**
  - VS Code est un éditeur extensible grâce aux extensions.
  - L’option **Add to PATH** permet d’ouvrir VS Code depuis le terminal avec `code .`
  - Il est recommandé de personnaliser votre éditeur selon votre stack de développement.
