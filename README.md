# DevSetup

DevSetup est un dépôt qui regroupe les étapes essentielles pour mettre en place un environnement de développement moderne. Il fournit une documentation simple et concise pour installer et configurer les outils indispensables avant de commencer un projet.

## Environnement de développement

Ce dépôt couvre l'installation et la configuration des outils suivants :

- **Visual Studio Code (VS Code)**
- **Git**
- **GitHub**

## Installation

Clonez le dépôt sur votre machine :

```bash
git clone https://github.com/dzeko003/DevSetup.git
```

Accédez ensuite au dossier du projet :

```bash
cd DevSetup
```

## Contenu du dépôt

- **creer-compte-github.md** : Guide pour créer un compte GitHub.
- **installation-git.md** : Guide d'installation de Git.
- **installation-VsCode.md** : Guide d'installation de Visual Studio Code.
- **.gitignore** : Fichier permettant d'ignorer certains fichiers et dossiers lors du versionnement Git.

## `.gitignore`

Le fichier `.gitignore` contient les règles suivantes :

```gitignore
# Ignorer les fichiers système
.DS_Store
Thumbs.db

# Ignorer le dossier node_modules
node_modules/
```

## Notes

- **Notes :**
  - Ce dépôt est destiné aux débutants souhaitant préparer leur environnement de développement.
  - La documentation est rédigée au format Markdown pour une lecture simple sur GitHub.
